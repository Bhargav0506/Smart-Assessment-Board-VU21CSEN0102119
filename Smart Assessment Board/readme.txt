Introduction
The Indian States and Capitals Quiz is a console-based application designed to help users learn and test their knowledge of the states and capitals of India. The quiz randomly selects states and prompts the user to enter the corresponding capital, providing immediate feedback on their answers.
Technologies Used
Programming Language: Python
Libraries: None (Standard Python libraries only)
Implementation Methodology
Data Structure: A dictionary is used to store the states as keys and their capitals as values.
Quiz Logic:
The program randomly selects a specified number of states from the dictionary.
For each state, it prompts the user to input the capital.
The user's answer is compared to the correct answer, and the score is updated accordingly.
User Interaction: The program provides feedback for each answer and displays the final score at the end of the quiz.
How to Use
Clone the Repository: Download or clone the repository to your local machine.
Run the Application:
Open a terminal or command prompt.
Navigate to the directory containing the quiz.py file.
Execute the command:
bash
python quiz.py

Answer the Questions: Follow the prompts to answer the questions. Type 'exit' to quit the quiz at any time.
View Your Score: At the end of the quiz, your score will be displayed, showing the number of correct answers out of the total questions asked.
Features
Random selection of states for each quiz attempt.
Immediate feedback for each answer.
Final score display at the end of the quiz.
